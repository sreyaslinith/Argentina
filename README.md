# 🇦🇷 Argentina World Champions Fan Website

A cinematic, interactive football fan website dedicated to the Argentina National Team and Lionel Messi. Built with HTML, CSS, and JavaScript, this project focuses on visual storytelling, animations, and immersive UI effects inspired by Argentina’s World Cup 2022 victory.

---

## 🌟 Live Experience

- 🎬 Cinematic loader intro ("Lights Off → Argentina 🇦🇷")
- ⚽ Full-screen hero stadium video background
- 🧊 Interactive flip-card squad system
- 🎥 Player highlight videos on card flip
- 🌫️ Blur effect when a player card is opened
- 🧑 Messi special cinematic story page
- 🎵 Argentina chant audio (Muchachos)
- 🖱️ Mouse spotlight effect on Messi page
- 🖼️ Auto slideshow memory wall
- 🏆 Animated trophy + floating stars effects

## 🎮 How It Works

### 🏠 Home Page (Hero Section)

* Starts with stadium background video
* Click anywhere to enter squad view

### ⚽ Squad Page

* Displays Argentina XI in a grid layout
* Hover → lift effect
* Click card → flips + plays video
* Click outside → closes card
* Messi card → redirects to Messi story page

### 🧑 Messi Page

* Cinematic intro sequence
* Background looping video
* Mouse-follow spotlight effect
* Auto-changing Messi photo gallery
* Animated trophy & stars
* Music toggle button for Argentina chant
